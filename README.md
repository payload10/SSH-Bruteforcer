
# SSH Bruteforcer

A SSH Buteforcer which will take the TARGET IP, USERNAME, PASSWORD LIST as input and then Bruteforce the input PASSWORD LIST to find the password of the USERNAME.


## Usage/Examples

python3 sshBruteforcer.py
